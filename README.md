### Morse Code Tools
A collection of educational resources about International Morse Code:
- [Morse Code Tools](https://siconghe.github.io/morse-code-tools/) - Complete reference guide with alphabet, numbers, and signals
- [Morse Code History](https://siconghe.github.io/morse-code-history/) - From the 1830s telegraph to modern applications
- [Free Online Translator](https://translatemorsecode.org) - Text-to-Morse and Morse-to-text with audio playback
